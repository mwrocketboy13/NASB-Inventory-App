# NASB-Inventory-App
Inventory app I created in python and turned into an exe

HOW TO SET UP THIS APP:
- DOWNLOAD ZIP FILE
- UNZIP FILES, YOU SHOULD SEE AN INTERNAL FILE, A .EXE, AND A .DB
- ADD A SHORTCUT FOR THE .EXE AND COPY IT TO THE DESKTOP

HOW TO USE THIS APP:
- OPEN APP
- SCAN BARCODE OR ENTER PART NUMBER AS "###"
- IF NEW PART, FILL OUT DATA FIELDS, SAVE
- IF EXISTING PART, ACQUIRE NECESSARY INFORMATION AND/OR DECREMENT QUANTITY
- LOCATION IS (CABINET)-(SECTION)-(ROW)-(COLUMN)
- REPEAT
